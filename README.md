# dotfiles :computer:

Familiar Dev Environment in any Linux server

## Prerequisite :traffic_light:

```bash
sudo apt install build-essential
```

## Install :beginner:

* Enter working directory and run:

```bash
make all
```

That is all you need to make any linux server your own.
